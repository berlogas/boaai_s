# ⚙️ Руководство администратора

## 👥 Пользователи

- Создавайте через Админ-панель (вкладка "Пользователи")
- Назначайте роли (admin/researcher)
- Редактирование и удаление пользователей доступно в той же вкладке

## 📚 Глобальная база знаний

### Загрузка файлов (упрощённый способ)

**Только администратор может загружать файлы в глобальную базу.**

1. **Скопируйте файлы** в папку для загрузки:
   ```
   /home/homo/projects/boaai_s/data_volume/uploads/
   ```
   
   **Поддерживаемые форматы (согласно PaperQA2):**
   - Научные статьи: `.pdf`
   - Текстовые файлы: `.txt`, `.md`
   - Веб-страницы: `.html`
   - Документы Office: `.docx`, `.xlsx`, `.pptx`
   - Файлы кода: `.py`, `.ts`, `.yaml`, `.json`, `.csv`, `.xml`

2. **Откройте Админ-панель**: http://localhost:8501 → вкладка **"⚙️ Admin"** → **"📚 Глобальная база"**

3. **Нажмите кнопку** "🔄 Загрузить файлы из папки uploads"

4. Файлы будут:
   - Обработаны и проиндексированы
   - Перемещены в хранилище глобальной базы
   - Добавлены в список документов

### Управление документами

- **Просмотр списка**: все загруженные документы отображаются в разделе "Документы в базе"
- **Удаление**: нажмите "🗑️ Удалить" в экспандере документа
- **Переиндексация**: кнопка "🔄 Переиндексировать" для пересоздания индекса (при проблемах с поиском)

### Журнал аудита

Во вкладке "📊 Аудит" отображаются все действия в системе:
- Создание/удаление пользователей
- Загрузка/удаление документов
- Переиндексация базы
